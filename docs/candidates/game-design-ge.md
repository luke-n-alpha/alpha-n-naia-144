# Game Design Candidate — GE (OC 대행)

**비고:** GE(Gemini) 할당 소진(429). OC가 GE 관점(시각/UX)에서 독립 후보 생성.

## 1. 게임 타이틀
- **한글:** 카페루아의 빛과 그림자
- **영문:** cafelua: Lights in the Jamo

## 2. 한 줄 시놉시스
사계절이 바뀌는 카페 cafelua에서 luke와 메이드 AI Alpha가 naia의 흔적을 따라 카페에 숨겨진 이야기를 조립하는 포토노벨 어드벤처.

## 3. 장르
포토노벨 + 탐색 어드벤처. 전투는 미니게임 수준의 리듬 퍼즐.

## 4. 핵심 메커니즘
카페의 각 구역을 방문해 사물을 조사하면, 한글 자모 타일로 구성된 "장면 기억"이 떠오른다. 플레이어는 떠오른 기억 조각(자모 밝기 배열)을 올바른 순서로 배치해 완전한 장면을 복원. Alpha와 대화하며 단서를 얻고, naia의 밝기 패턴을 해독해 다음 구역의 단서를 연다.

## 5. 화면 구성
- `SCENE_TITLE`: 계절별 배경 자모 패턴 + 타이틀 + 메뉴
- `SCENE_CAFE`: 1층 카페 탐색. 카운터/피아노/소파/창가를 방향키로 이동. 각 위치에서 조사/대화 선택.
- `SCENE_ATELIER`: 2층 아틀리에. 자모 배열 퍼즐 메인 무대. 복원된 기억이 벽에 걸린 그림처럼 표시됨.
- `SCENE_DIALOG`: 하단 대사창 + 화자 이름 + 감정 아이콘(텍스트). 선택지는 최대 3개.
- `SCENE_BATTLE`: 리듬 퍼즐. 밝기 파형이 좌→우로 흐르고, 타이밍에 맞춰 확인키. 정확도가 복원률에 반영.
- `SCENE_ENDING`: 복원된 장면들의 콜라주 + 결말 텍스트.

## 6. 챕터 구조
1. **봄의 카페 개점**: 기본 조작 학습 + 카페 둘러보기 + 첫 밝기 이상 발견
2. **여름의 기억**: Alpha와 대화 깊어짐 + naia 첫 등장(밝기 패턴) + 1층 퍼즐
3. **가을의 아틀리에**: 2층 진입 + 핵심 기억 복원 퍼즐 + Alpha 감정 변화
4. **겨울의 새벽**: 최종 전투(리듬 퍼즐) + 엔딩 분기

사계절이 챕터와 1:1 매핑. 시각적으로 배경 밝기 팔레트가 변해서 진행 상황 직관적.

## 7. 캐릭터 상호작용
- luke: 침착한 관찰자. 플레이어의 선택이 luke의 태도를 결정.
- Alpha: 감정 학습 곡선. 초기엔 정중하고 기계적, 진행할수록 감정 표현이 풍부해짐. emotion_id로 시각 표현.
- naia: 말 없는 안내자. 밝기 패턴으로 방향 제시. 해독 성공 시 짧은 묘사 텍스트로 반응.

## 8. 전투 시스템
적 처치가 아닌 리듬 정렬. Pixel mode에서:
- 자모 밝기 파형이 화면을 가로지름
- 플레이어가 타이밍에 맞춰 확인키 입력
- 정확도 → 복원 게이지 충전
- 게이지 100% → 장면 안정화 → 다음 챕터

난이도는 챕터 진행에 따라 파형 속도 증가.

## 9. 엔딩 조건
멀티 엔딩 3종:
- **봄의 재개**: 모든 기억 복원 + Alpha 신뢰도 높음 → 따뜻한 일상 회복
- **별빛 기록**: naia 복원률 높음 + 특정 선택지 → naia와의 특별한 순간
- **미해결의 새벽**: 복원 미완료 → 여운이 남는 열린 결말

## 10. 플레이 타임 예상
1회차 15-20분. 전체 엔딩 컬렉션 40-60분.
