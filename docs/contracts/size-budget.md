## 추적 규칙

1. 모든 모듈 완료 후 `size` + `strip` 후 파일 크기 기록
2. 예산 초과 시 제약 검사에서 자동 거부
3. 최종 목표: 총 ≤ 1,400 KiB (40 KiB 여유)

## 측정 방식

```bash
gcc -Os ... -o build/game && strip build/game && ls -la build/game
size build/game
```
