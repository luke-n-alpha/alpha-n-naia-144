**상태:** v1.1 (CX 리뷰 반영)
**변경:** 자모 퍼즐 규칙 구체화, 장면 전이 맵 추가, 콘텐츠 예산 가드 추가

## 1. 타이틀

- **한국어:** 자모등 아래의 나이아
- **English:** Naia Beneath the Jamo Lights

## 2. 시놉시스

luke가 운영하는 카페 cafelua에서 하루 영업을 마무리하던 밤, 공간을 구성하는 한글 자모 타일의 밝기 배열이 무너지기 시작한다. 메이드 AI Alpha와 함께 naia가 남긴 밝기 흔적을 따라 카페에 숨겨진 이야기를 복원하는 따뜻한 미스터리.

## 3. 장르

텍스트 어드벤처 + 탐색 퍼즐 + 리듬 미니게임.

## 4. 핵심 메커니즘

플레이어는 80×25 카페 화면을 방향키로 이동하며 조사·대화·선택을 반복. 각 구역의 단서는 한글 자모 밝기 패턴으로 표현되며, 플레이어가 올바른 순서로 밝기를 정렬해 장면 기억을 복원. 진행은 턴제.

### 4.1 자모 퍼즐 규칙 (Text mode)

퍼즐은 **분해된 자모 타일(jamo tile)** 단위로 동작. 한글 음절이 아닌 개별 초성·중성·종성이 80×25 그리드의 각 셀에 배치됨.

**기본 동작:**
- 화면에 3-5개의 자모 타일이 무작위 밝기로 표시됨
- 플레이어는 커서(방향키)로 타일을 선택하고, 확인키로 밝기 순환(0→1→2→...→15→0)
- 목표: 모든 타일의 밝기를 정답 배열에 맞춤
- 정답은 naia의 밝기 흔적(섞여서 표시됨)을 참조

**해결 상태:** 모든 타일의 밝기가 정답과 일치 → 장면 복원 이벤트 발생

**상태 저장:** 퍼즐 상태는 각 타일의 (자모 인덱스, 현재 밝기) 배열로 표현. save/load 시 저장.

### 4.2 리듬 퍼즐 규칙 (Pixel mode)

**기본 동작:**
- 화면 하단에 4개 레인(각 레인 = 한글 자모 하나)
- 각 레인에서 밝기 블록이 좌→우로 일정 속도로 이동
- 블록이 판정선에 도달할 때 확인키 누르면 판정 (PERFECT/GOOD/MISS)
- PERFECT → 복원 게이지 +3, GOOD → +1, MISS → 0
- 게이지 100% → 장면 안정화

**난이도:** 챕터당 블록 수 Ch1=8, Ch2=12, Ch3=16, Ch4=24. 속도는 Ch당 10% 증가.

## 5. 화면 구성

| Scene | 모드 | 설명 |
|-------|------|------|
| TITLE | Text | 계절별 자모 패턴 배경 + 타이틀 + 세이브 3슬롯 |
| CAFE | Text | 1층 탐색. 카운터·피아노·소파·창가. 방향키 이동, 조사/대화 선택 |
| ATELIER | Text | 2층 아틀리에. 자모 배열 퍼즐 메인. 복원된 장면이 시각적으로 누적 |
| DIALOG | Text | 하단 대사창 + 화자명 + emotion_id. 선택지 최대 3개 |
| BATTLE | Pixel | 리듬 정렬 퍼즐. 4레인, 밝기 블록 이동, 타이밍 판정 |
| ENDING | Text | 복원 장면 콜라주 + 결말 분기 |

### 5.1 장면 전이 맵

```
TITLE ──→ CAFE (새게임) / DIALOG (이어하기)
CAFE ──→ DIALOG (NPC 조우)
CAFE ──→ ATELIER (2층 계단 발견 후)
DIALOG ──→ CAFE (대화 종료)
DIALOG ──→ BATTLE (기억 불안정 이벤트)
ATELIER ──→ DIALOG (기억 복원 시)
ATELIER ──→ BATTLE (Ch3-4 퍼즐)
BATTLE ──→ DIALOG (안정화 후)
BATTLE ──→ ENDING (Ch4 클리어 시)
ENDING ──→ TITLE
```

## 6. 챕터 구조

사계절 = 챕터. 배경 팔레트가 봄→여름→가을→겨울로 변화.

| Ch | 계절 | 제목 | 목표 |
|----|------|------|------|
| 1 | 봄 | 마감 후의 카페 | 기본 조작 학습, 카페 탐색, 첫 밝기 이상 발견 |
| 2 | 여름 | Alpha의 기록 | Alpha와 대화 심화, naia 첫 등장(밝기 패턴), 1층 퍼즐 |
| 3 | 가을 | 2층 아틀리에 | 2층 진입, 핵심 기억 복원 퍼즐, Alpha 감정 변화 |
| 4 | 겨울 | 새벽의 빛 | 최종 리듬 퍼즐, naia 추적, 엔딩 분기 |

## 7. 캐릭터

| 캐릭터 | 역할 | 표현 |
|--------|------|------|
| luke | 관찰자·선택 주체 | 플레이어 = luke |
| Alpha | 감정을 배우는 메이드 AI | 초기 기계적 → 점차 감정 표현. emotion_id로 시각화 |
| naia | 말 없는 안내자 고양이 | 밝기 패턴으로 방향 제시. 해독 시 짧은 묘사 텍스트 |

## 8. 엔딩

멀티 엔딩 3종. Alpha 신뢰도, naia 복원률, 대화 태도에 따라 분기:

1. **일상 회복**: 모든 기억 복원 + Alpha 신뢰도 높음 → 따뜻한 일상
2. **기억 공명**: naia 복원률 높음 + 특정 선택 → naia와의 특별한 순간
3. **미해결 새벽**: 복원 미완료 → 여운이 남는 열린 결말

## 9. 플레이 타임

- 1회차: 20-25분
- 전체 엔딩 컬렉션: 40-60분

## 10. 콘텐츠 예산 가드

| 항목 | 제한 | 근거 |
|------|------|------|
| 챕터당 대사 라인 | ≤ 150줄 | 4챕터 × 150 = 600줄 × ~40바이트 = ~24KB |
| 전체 대사 | ≤ 800줄 | 3엔딩 분기 포함 |
| 퍼즐 데이터 | ≤ 2KB/챕터 | 타일 배치 + 정답 배열 |
| 엔딩 텍스트 | ≤ 50줄/엔딩 | 3엔딩 × 50줄 = 150줄 |
| **시나리오 총계** | **≤ 100KB** | size-budget 150KB 여유 중 100KB |

## 11. 구현 우선순위

1. Text mode 탐색 + 대화 (Ch 1-2)
2. 자모 밝기 시각화
3. Pixel mode 리듬 퍼즐 (Ch 3-4)
4. 세이브/로드
5. 엔딩 분기
