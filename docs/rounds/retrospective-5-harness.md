- 역할을 문서로만 적어놓고 강제할 수단이 없었다

- 회고를 안 해도 아무 시스템이 막지 않았다

## 무엇을 고쳤나

1. `harness/roles/OC.md` — OC 역할 정의. 코드 작성 금지 명시
2. `harness/roles/CX.md` — CX 역할 정의. 시스템 코드 생성 + 리뷰
3. `harness/roles/GE.md` — GE 역할 정의. 비주얼/UX + 시나리오
4. `harness/spec.toml` — 전체 파이프라인, 게이트, 폴백 체인 정의
5. `harness/run.py` — 런타임. 게이트 강제, 회고 체크, human 승인 강제
