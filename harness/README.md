# Harness Runtime

에이전트 조직이 사람 개입 없이 돌아가도록 강제하는 런타임.
프로세스 = 산출물. 코드가 곧 프로세스다.

## 실행

    python3 harness/run.py --phase <phase_name>

## 역할 파일

- roles/OC.md — 오케스트레이터. 코드 작성 금지.
- roles/CX.md — 시스템 코드 생성 + 리뷰.
- roles/GE.md — 비주얼/UX + 시나리오 생성 + 리뷰.

## 규칙

1. 각 에이전트는 역할 파일에 정의된 것만 한다
2. 게이트 통과 없이 다음 phase로 넘어갈 수 없다
3. 막히면 리소스를 끌어온다 (다른 에이전트, 외부 문서, human)
4. 회고 없이 phase를 닫을 수 없다
5. human 승인 없이 인도할 수 없다

## 폴백 체인

에이전트가 응답 불가 시:
1. 3회 재시도 (간격 30s)
2. 동일 모델 다른 프롬프트로 가상 에이전트 생성
3. R < 2면 human 개입 강제 (자동 진행 금지)
